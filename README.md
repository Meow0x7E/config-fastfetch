## Installation

You can set up the configuration using either of the following methods

### Manual Download

1. Download [config.jsonc](config.jsonc)
2. Place it in the `$HOME/.config/fastfetch` directory.

### Run Command

Execute the following command to apply the configuration automatically

```bash
[[ -d "$HOME/.config/fastfetch" ]] || mkdir "$HOME/.config/fastfetch"
curl -L -S -s -O --output-dir "$HOME/.config/fastfetch" https://raw.githubusercontent.com/Meow0x7E/config-fastfetch/main/config.jsonc
```
